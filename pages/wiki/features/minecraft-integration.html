Minecraft Integration

With Wadder, you can integrate your Minecraft server to your Discord server and keep track of the players online. Here's how to set it up:

Step 1: Add the Wadder bot to your Discord server

If you haven't already, invite the Wadder bot to your Discord server by clicking on the link below:

Invite Wadder to your server

Step 2: Set up the server information

Once the bot is added to your Discord server, you need to set up the server information. Here's how:

  1. Open your Minecraft server console and find the IP address and port number of your server.
  2. Open your Discord server and type the following command in any text channel: /setup_server [ip address] [port number] [channel] [server name] [server description]
  3. Replace the placeholders with your own information. For example: /setup_server 127.0.0.1 25565 #minecraft-server "My Minecraft Server" "A cool Minecraft server"
  4. The server information will be displayed in the specified channel and refreshed every 5 minutes.

Step 3: Watch the server information

You can now watch the server information in the specified Discord channel. The bot will refresh the information every 5 minutes.

Note: Wadder does not keep or take any information from your server.
